# PGMMV Snap-To-Tile Plugin

[![CI](https://github.com/kidthales/pgmmv-snap-to-tile-plugin/actions/workflows/ci.yml/badge.svg)](https://github.com/kidthales/pgmmv-snap-to-tile-plugin/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

Snap-to-tile action commands for [PGMMV](https://rpgmakerofficial.com/product/act/en/manual/01_01.html).

| Action Command            | Description                                                                                                                                                                                                                                                    |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Snap to Tile              | Snap object position to current tile. Tile origin (0,0) is top left, (1,1) is bottom right, and (0.5, 0.5) is center (default).Snap object position to current tile. Tile origin (0,0) is top left, (1,1) is bottom right, and (0.5, 0.5) is center (default). |
| Snap to Tile (Horizontal) | Snap object position horizontally to current tile. Tile origin 0 is left, 1 is right, and 0.5 is center (default).                                                                                                                                             |
| Snap to Tile (Vertical)   | Snap object position vertically to current tile. Tile origin 0 is top, 1 is bottom, and 0.5 is center (default).                                                                                                                                               |
